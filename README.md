
# Supermarket PL/SQL Project

## Project Description
A complete PL/SQL implementation of a supermarket inventory and billing system demonstrating Oracle PL/SQL concepts.

## Concepts Demonstrated
- **Records**: Product information storage
- **Collections**: All three types (VARRAY, Nested Table, Associative Array)
- **GOTO Statements**: Control flow for discount calculations

## How to Run
1. Open Oracle SQL Developer
2. Connect to your database
3. Run `scripts/supermarket_simple.sql`
4. View output in DBMS Output window
